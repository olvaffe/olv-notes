PHP
===

## Installation

- install php
  - `pacman -S php`
- install composer
  - `pacman -S composer`, or
  - `php -r "copy('https://getcomposer.org/installer', 'composer-setup.php');"`
  - `./composer-setup.php`
- using composer
  -`composer install`
